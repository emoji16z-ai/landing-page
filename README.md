### Clone to local

If you prefer to do it manually with the cleaner git history

```bash
npx degit MoseStudio/remox-lite my-app
cd my-app
pnpm install
```

## Usage

### Development

Just run and visit http://localhost:5173

```bash
pnpm run dev
```

### Build

To build the App, run

```bash
pnpm run build
```
